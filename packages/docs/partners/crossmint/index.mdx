NFT infrastructure for developers.

## About Crossmint

Crossmint provides APIs and tools to embed blockchain capabilities into applications without requiring cryptocurrency expertise. Key offerings include:

- **Embedded Wallet Infrastructure**: Custodial and non-custodial wallets supporting 40+ chains
- **NFT Minting Suite**: APIs for creating/dynamic NFTs across 20+ blockchains
- **Payment Gateway**: Fiat/crypto checkout with 95%+ approval rates in 195+ countries
- **Verifiable Credentials**: On-chain identity management at scale

Notable clients include Santander Bank, Mastercard, NBC Universal, and Coinbase. The platform serves over 40,000 enterprises and developers, positioning itself as critical infrastructure bridging Web2 and Web3.

---

## Technology

Crossmint's stack combines enterprise security with developer-friendly abstractions:

- **MPC Vaults**: Multi-party computation wallets with Fireblocks-grade security
- **Gas Station Architecture**: Automatic fee handling using fiat billing
- **Universal Smart Contracts**: Pre-audited templates for NFTs/credentials
- **Chain-Agnostic SDKs**: Single API integration for EVM/Solana/Aptos/Sui

Key innovations include:

1. Email-based NFT delivery with auto-generated wallets
2. Cross-chain swap infrastructure
3. Dynamic NFT updating via REST API
4. Fraud prevention systems with 0.03% chargeback rate

## Key Features

## Integration with Eliza

While no official ElizaOS plugin was found in public repositories, Crossmint's architecture suggests multiple integration pathways:

- **REST API Endpoints**: Direct API calls for wallet/NFT operations from Eliza workflows
- **Webhook Triggers**: Automate Eliza actions based on on-chain events
- **Shared Auth Layer**: Potential SSO integration using Crossmint's Stytch-based auth
- **Joint Use Cases**:
  - Eliza-powered AI agents managing Crossmint wallets
  - Automated NFT minting via Eliza workflow builder
  - Cross-platform credential verification systems
